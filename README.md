# GengoSync: Smart Vocabulary Integrator

**Проблема:** Користувачі часто забувають перекладені в Google Translate слова одразу після закриття вкладки, а процес ручного перенесення лексики в окремий зошит чи додаток є надто часозатратним.

**Рішення:** Веб-застосунок на Java, що інтегрується з інтерфейсом перекладача через браузерне розширення, дозволяючи зберігати нові слова та їх переклади до персонального словника одним кліком.

**Цільовий користувач:** Студенти, лінгвісти та спеціалісти, які активно працюють з іншомовним контентом і прагнуть ефективно накопичувати власний словниковий запас швидко та зручно.
